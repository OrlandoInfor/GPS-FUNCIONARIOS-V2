# Instruções do Projeto — GPS Funcionários

Este arquivo contém as diretrizes de fluxo de trabalho para o projeto de rastreamento.

## 💾 Protocolo de Auto-Save (Obrigatório)

Sempre que uma tarefa for concluída ou um progresso significativo for feito neste projeto, o agente **DEVE** atualizar o arquivo central de sessão:
- **Caminho:** `/Users/orlandogarcia/.opencode/GPS-FUNCIONARIOS-SESSAO.md`

### O que atualizar:
1. **Status de Tarefas:** Marcar como concluído [x] na seção "O QUE JÁ FOI FEITO".
2. **Novas Demandas:** Adicionar novos itens à seção "PRÓXIMOS PASSOS".
3. **Mudanças Técnicas:** Atualizar a seção "ARQUITETURA" ou "MELHORIAS" se houver mudanças no código ou banco de dados.
4. **URLs/Credenciais:** Manter a tabela de URLs e credenciais sempre atualizada.

## 🚀 Como Retomar
Ao iniciar uma sessão, se o usuário mencionar "gps funcionario", o agente deve ler o arquivo de sessão acima para obter o contexto completo antes de sugerir ações.
